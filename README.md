# 项目介绍
  本项目用户快速搭建小程序开发基本模板, 以及介绍整理小程序开发中常用的组件。

# 环境依赖
  微信开发环境。

# 目录结构描述
    ├── ReadMe.md           // 帮助文档
    ├── pages               // 小程序基本页面
    │    ├── components     // UI组件库
    │    ├── index          // 一键登录页面
    │    └── home           // 主要组件展示页面
    ├── utils               // 工具类脚本
    ├── app.js              // 小程序入口文件
    ├── app.json            // 小程序配置文件
    ├── app.wxss            // 小程序基本样式文件
    ├── ...
